Télécharger python sur windows ?

Pour installer la dernière version de python sur votre ordinateur
rendez-vous sur https://www.python.org/downloads/ et de cliquer sur download.
Une fois python installer, éxécuter le fichier et suivez les instructions. Une fois terminé, fermer la fenêtre.

Et voila python est installé !

Comments démarrer des fichiers python sur windows ?

Pour démarrer un fichier python sans bugs, commencer par ouvrir l'invite de commande, ensuite tapez :

cd "Chemin d'accés au fichier"

Puis :

py "Fichier.py"

Et voila vous avez démarrer votre fichier !
